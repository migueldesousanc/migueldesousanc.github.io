001
Automatic Classification of Antarctic Imagery
Master's dissertation. Processed Worldview-2 and Landsat-8 imagery to create vegetation maps across ice-free areas of the Antarctic Peninsula. Implemented and benchmarked SVM, Random Trees, and kNN algorithms for classification accuracy optimisation.
Python SVM Landsat-8 Worldview-2 ML
002
Automated Mineral Deposit Analysis
Built a custom ArcPy script integrated into ArcGIS Pro Notebooks to automate analysis of mineral deposit data across Africa. The pipeline summarises, classifies and ranks features by country and mineral type — identifying regions with the highest exploitation concentration.
ArcPy ArcGIS Pro Python Automation
003
Precision Agriculture & Vineyard Monitoring
Performed vegetative health monitoring for vineyards using NDVI, NDWI, and Water Stress indices derived from Drone and Sentinel-2 data. Processed imagery with Agisoft Metashape and QGIS. Also conducted a geological and environmental impact study for the Pampilhosa Lime Kilns.
NDVI/NDWI Sentinel-2 Drone QGIS Metashape
